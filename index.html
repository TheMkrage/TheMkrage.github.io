Meet the Bots

We're fascinated with robots because they are reflections of ourselves.

Gateway

Gateway was a robot built for the 2014 FRC game Aerial Assist. Because I was just a freshman when Gateway was created, I only worked to assemble Gateway.

Awards: - Engineering Inspiration** (Los Angeles Regional)

Robie

Robie was a robot built for the 2014-2015 VEX game Skyrise (our rookie vex year). Robie was the first robot that I worked to program. Through programming Robie, I learned the basics of Proportional Integral Derivative Controllers. Robots. Robie taught me skills that have been applicable to almost every other project I have worked on.
Awards:
- 2x Excellance Award*
- 2x Tournement Champions*
- 2x Think Award
- 2x Programming Skills Winner
- 2x Robot Skills Winner
- 1x Innovate Award
- Robot Skills State Champions**
- State Championship Amaze Award
- Quarterfinalists at State Championships

ArisTOTEle

ArisTOTEle was a robot built for the 2015 FRC season Recycle Rush. ArisTOTEle was the first FRC Rrobot I programmed. Some main features of ArisTOTEle include 2 seperate PID loops tuned to the same speed that allow two intakes on seperate rails to move together, an autonomous that scores the max amount of points, and a drive equation that uses angular velocity and PID to turn quickly.
Awards:
- Excellance in Engineering (Los Angeles Regional)
- Quarterfinalists (Los Angeles Regional)
- Innovation in Control (Los Vegas Regional)
- Tournement Finalists** (Los Vegas Regional)

* Qualified for State Champs (Vex Only)
**Qualified for World Champs
